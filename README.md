# 省钱.skill

> 把一句模糊的省钱需求，变成一份「直接照做」的可执行方案——去哪个平台、搜什么词、花多少钱、第一步点什么，全写好，还带可点搜索直链。
> 支持：WorkBuddy、Claude Code、Codex，以及其他支持 Skills 的 Agent。零代码，一个 SKILL.md 即可运行。

## 省钱解决什么问题

- **信息差难捞**：同物在不同消费层级价差巨大，但普通人不知道去哪找「卖白菜价的特定人群段」。
- **比价动脑多**：市面工具只给信息，不给你「直接照做」的三步动作。
- **省钱无正反馈**：省了也记不住、难坚持——缺一个把省下的钱累积起来看见的钩子。
- **平替知识散落**：零散好价不系统、不可复用，每次都得重新搜。

省钱用一套**「洼地三问法」**作方法论脊柱，而不是零散好价堆砌：

> **渠道洼地（去哪买最便宜） ＋ 时段人群（谁在何时甩货） ＋ 精准搜索词（怎么捞） ＝ 把溢价挖掉**

同一件商品，在不同人群、渠道、时段的售价能差出几倍——这是普遍存在的「价格错配」。省钱的本质是找到价格洼地：把货卖白菜价的「特定人群 + 特定时段 + 低价渠道」，用精准搜索捞出来。

## 快速开始

安装完成后，直接在 Agent 中输入你的省钱需求：

```
省钱 我想买 iPhone 但只有 3000 块，怎么买最划算？
```

它会读取你的需求，给出「直接照做」的三步方案（去哪个平台 + 搜什么词 + 预期价 + 第一步），并附可点搜索直链。

已知具体需求时，也可以显式调用：

```
@省钱 我每月外卖花太多，怎么省？
/省钱 200 块想穿出老钱风，怎么买？
```

对每个需求，产出一份结构化 Markdown 报告：**直接照做**（含可点搜索直链）→ 一句话背景 → 平替对照 → ≤3 条避坑 → **省钱账本**（本次省 / 累计省，跨会话自动累计于 `~/.shengqian-junshi/账本.md`；该文件含你的消费主题，属本地隐私，请勿纳入仓库或截图对外分享）。

## 能力一览

覆盖 10 大类高频消费场景：**吃、穿、住、行、用（数码/家电）、玩、学、金融/理财、日用/订阅审计、健康（仅安全项）**。

| 类目 | 代表性省钱手法 | 典型省幅 |
|---|---|---|
| 吃 | 连锁代下单 / 超市鲜食清仓 / 1688 进货自做 | 40–70% |
| 穿 | 1688 产业带同款 / 跟模特店 / 内增高工厂 | 50–70% |
| 住 | 原住民闲置房 / 青旅包月 / 家电以旧换新 | 30–60% |
| 行 | 0 元卡位租车 / 廉航错峰 / 顺风车·积分 | ≈50%+ |
| 用（数码/家电） | 美版二手 / 组装台式 / 游戏低价区 | 30–50% |
| 玩 | 官方家庭组 / 低成本社交 / 免费替代 | 合规 |
| 学 | 巨头免费课 / 资料平替 / 图书馆 | 100% |
| 金融/理财 | 高息活期 / 免年费卡 / 不套现不分期 | 守住 |
| 日用/订阅 | 大包装 / 取消闲置会员 / 官方家庭组 | 显著 |
| 健康（仅安全项） | 校医务室 / 社保报销 / 老国货药 | 低费 |

所有具体价格均标「以实时搜索为准」，不承诺写死数字；违禁药自用、退货滥用等违规/伤身项已从知识库剔除。代下单、合租、低价区等灰色地带仅作信息参考，并附辨别指南：认准高好评率与真实带图评价、警惕刷分刷评、只走平台担保、绝不预付款。

## 安装

### WorkBuddy 与其他支持 Skills 的 Agent

在终端执行：

```bash
npx -y skills add jelly-000/shengqian-junshi -g
```

安装后回到 Agent，输入 `/省钱 新手入门` 即可开始。

### Claude Code / Codex

将仓库克隆到对应技能目录：

```bash
git clone https://github.com/jelly-000/shengqian-junshi.git ~/.claude/skills/省钱
git clone https://github.com/jelly-000/shengqian-junshi.git ~/.codex/skills/省钱
```

### 更新

已安装时，对当前 Agent 说「更新省钱」，它会同步官方仓库，不会修改你在 `~/.shengqian-junshi/` 中的账本与记录。
