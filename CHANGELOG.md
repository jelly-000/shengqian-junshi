# Changelog

本项目所有重要变更记录于此。格式参考 [Keep a Changelog](https://keepachangelog.com/)。

## v1.0.0 — 2026-07-21

首个开源版本。基于「洼地三问法」的方法论脊柱，经多轮实战验证与商业诊断迭代成型。

### 核心能力
- 洼地三问法脊柱：渠道洼地 ＋ 时段人群 ＋ 精准搜索词
- 10 大类覆盖：吃 / 穿 / 住 / 行 / 用 / 玩 / 学 / 金融 / 日用 / 健康(仅安全项)
- 报告模板：直接照做(含可点搜索直链) → 背景 → 平替对照 → 避坑 → 省钱账本
- 知识库 30+ 条策展平替案例，全部带时效标签
- KB 自更新（生成报告时按需抽查，价偏离 >20% 回填，非后台定时任务）

### 迭代历程
- **P0**：原创策展(知识为公开可验证的平替案例) + 转公开(visibility → public)
- **P1**：可点搜索直链 + 省钱账本 + 口语化触发词(这个好贵/有便宜的没/太贵了)
- **P2**：开源分发结构(README + examples/ 4 份实战报告 + 开源仓库页呈现)
- **P2+**：能力打磨(洼地三问法脊柱 / 可点直链 / 省钱账本 / skill 形态)
- **P3**：省钱账本跨会话持久化(单 append-only 文件 `~/.shengqian-junshi/账本.md`，无数据库)

### 实战验证
4 个真实场景自检通过：
- iPhone 3000 元拿下(数码大件洼地，省 ~¥1900 vs 全新入门新机)
- 外卖每月花太多(高频开支多路径，月省 ¥600–900)
- 自装 1440p 游戏主机(组装洼地，2026 内存通胀下守预算，省 ~¥2000 vs 同性能全新整机)
- 200 元老钱风整身(穿搭平替，省 ~¥250–400 vs 优衣库全新同风格)

### 安全边界
违规/伤身项(违禁药自用、退货滥用等)已从知识库剔除；所有价格标「以实时搜索为准」。
